1.- Para instalar el proyecto se debe crear una base de datos en phpmyAdmin de preferencia, con el nombre 'battles'.
2.- Ejecutar las migraciones de laravel con el comando 'php artisan migrate'.
3.- Ejecutar los seeders con el comando 'php artisan db:seed'.
4.- Dirigirse a la url: http://localhost/battles/ para verificar que todo funcione, si es así deberá mostrarse una pantalla con las imagenes de los guerreros.
5.- Elegir un guerrero y pulsar el boton 'Batalla' para buscar un peleador al azar.
6.- Una vez seleccionados aparecerá una ventana con los dos guerreros, pulsar el boton 'Pelear' y esperar al ganador.
7.- Aparecerá una alerta con los resultados de la batalla
